# Driver Guide

## Before work

1. Open Saki Operations while online.
2. Confirm your name and role.
3. Check Home shows Network Online and no failed sync items.
4. Confirm camera permission when requested.
5. Install the PWA from the approved company link for reliable full-screen access.

## Offline use

Operational work can continue offline. Keep the app installed and avoid clearing browser storage. When connectivity returns, leave the app open so queued work can upload.

## Updates

When **Update available** appears, complete the current operation first, then apply the update. After restart, Settings About shows the running version and build.

## Reporting a problem

1. Note the Error ID if one appears.
2. Open Settings.
3. Select Copy System Information.
4. Send the copied information with a short description of what happened.

Do not include passwords, refresh tokens, or customer information in support messages.
